## Overview:
This math project using Numpy and PyQt5 is a simple made application
to calculate math functions such as the Iterative Power function.

### Options

> Iterative Power
> Inverse Power


### Built With

* [Python](https://www.python.org)
* [PyQt5](https://pypi.org/project/PyQt5/)
